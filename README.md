## 🛠 How to Run Locally

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Ueslly-Code/my-portfolio.git
   ```
2. **Move into the project folder:**
   ```bash
   cd my-portfolio
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
